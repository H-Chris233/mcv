# Security Policy

This project is experimental and unaudited. Do not use it as the sole storage for irreplaceable secrets.

## Supported Versions

No production version is currently supported. The repository is in M0 skeleton stage.

## Reporting Vulnerabilities

Until a public reporting channel is configured, open a private security advisory on the repository host or contact the maintainer directly.

Do not include user passwords, key material, Shamir shares, vault plaintext, complete vault blobs, or complete NFC payloads in reports.

## Out Of Scope

- Cloning access cards.
- Bypassing access control systems.
- Reading or modifying unauthorized third-party cards.
- Guarantees on rooted or fully compromised Android devices.
- Protection against advanced physical or side-channel attacks.
